# pinafore-hms
Pinafore HMS, open source hotel management software based on web2py

## Roadmap

**v. 0.1.0**

* Basic database model
* Placeholder controllers

**v. 0.2.0**

* Basic forms for reservations
* Basic layout.html
* Navigation menu and sidebar

**v. 0.3.0**

* Custom forms

**v. 0.4.0**

* Custom layout.html

**v. 0.5.0**

* Guest search
* Rate planning
* Housekeeping status

**v. 0.6.0**

* Basic reports:
  * In-house
  * Expected arrivals
  * Expected departures
  * Housekeeping status

**v. 1.0.0**

* Initial release
* Most documentation available
* Should be basically functional:
  * Reservations
  * Check-in/out
  * Housekeeping status
  * Rate plans
  * Posting charges/adjustments/settlements
  * Guest search by absolute value
  * Some reports
